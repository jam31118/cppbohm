# Propagator of Bohmian particles in C++

A set of propagator objects which propagate Bohmian particles along with corresponging wavefunction propagated by time-dependent Schr√∂dinger equation.


# Installation

### Install dependencies

- LAPACK
- matrix

### Set envrionment variables

``` bash
export LAPACKHOME=/path/to/lapack
export MATRIXHOME=/path/to/matrix
```

### Build

``` bash
make
```
